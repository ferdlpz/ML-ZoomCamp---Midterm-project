# Term deposit prediction

## Introduction
This project was undertaken as part of the Midterm Project Assignment for the Machine Learning Zoomcamp 2025. The dataset has taken from Kaggle on this link (https://www.kaggle.com/datasets/rashmiranu/banking-dataset-classification?select=new_test.csv)

Problem:
A Portuguese Bank has experienced a decline in revenue. Analysis of this situation identified a critical root cause: the low rate of customer investment in long-term deposits.
